# IT Asset Management System

## Overview

The IT Asset Management System is a web-based application developed for CMPDI to efficiently manage, monitor, and maintain organizational IT assets across multiple locations.

The system centralizes asset information, employee allocation, warranty monitoring, QR code generation, survey lot creation, and dashboard analytics into a single platform.

---

## Features

### Dashboard Analytics

- Real-time asset statistics
- Asset category distribution charts
- Asset status monitoring
- Warranty expiry monitoring
- Recent asset activity

### Multi-Location Management

- Location selection after admin login
- Location-based asset filtering
- Current location indicator
- Location-specific dashboards

### Asset Management

- Register and manage IT assets
- Update asset details
- Delete assets
- Search assets

### Product Management

- Add and manage products
- Store purchase and warranty details
- Upload supporting documents and images

### Employee Management

- Maintain employee records
- Allocate assets to employees
- Track asset ownership

### Warranty Monitoring

- Detect expired warranties automatically
- Dedicated warranty report page
- Dashboard warranty indicators

### QR Code Management

- Generate QR labels
- Print asset labels
- Quick asset identification

### Survey Lot Management

- Auto-generate lot numbers
- Group eligible assets
- Bulk asset assignment
- Survey and disposal support

### Reports

- CSV export
- PDF export
- Asset reports
- Survey reports
